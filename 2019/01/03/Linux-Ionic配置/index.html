Linux Ionic配置

  • Linux Ionic
  • 2019-01-03

查看系统版本

uname -a

查看系统位数

uname -m

如果是x86_64就是64位，如果是i686就是32位

安装nodejs

  1. 进入Nodejs官网下载安装包
  2. 解压安装包
    cd Downloads
tar -xvf node-v10.14.2-linux-x64
  3. 将该目录移动到通用软件目录
    sudo mv node-v10.14.2-linux-x64 /opt/
  4. 安装 npm 和 node 命令到系统命令
    sudo ln -s /opt/node-v6.10.3-linux-x64/bin/node /usr/local/bin/node 
sudo ln -s /opt/node-v6.10.3-linux-x64/bin/npm /usr/local/bin/npm
  5. 运行npm -v、node -v查看是否安装成功

搭建JAVA环境

  1. 下载jdk-8u191-linux-x64.tar.gz
  2. 解压
    tar -zxvf jdk-8u191-linux-x64.tar.gz
  3. 移动到opt目录下
    sudo mv jdk1.8.0_191 /opt/

搭建ANDROID环境

  1. 下载android-sdk_r24.4.1-linux.tgz
  2. 解压
    tar -zxvf android-sdk_r24.4.1-linux.tgz
  3. 移动到opt
    sudo mv android -sdk-linux /opt
  4. 更新（会获取到目前所有的安卓版本，文件会有点大）
    android-sdk-linux/tools/android update sdk --no-ui

配置环境变量

  1. 编辑profile文件
    sudo vim /etc/profile
  2. 在profile文件后加入如下代码
    export JAVA_HOME=/opt/java/jdk1.8.0_191
export ANDROID_HOME=/opt/android-sdk-linux
export PATH=$JAVA_HOME/bin:$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools:$PATH

安装cordova ionic

npm install -g cordova ionic

完成后运行cordova、ionic查看是否安装成功
如果安装太慢，可以设置淘宝镜像安装，或是使用cnpm安装

npm config set registry https://registry.npm.taobao.org
